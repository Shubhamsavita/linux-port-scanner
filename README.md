# Linux Network Port Scanner

## Description

A Python-based TCP Port Scanner that scans common ports on a target host.

## Features

### Version 1
 - Scan Common TCP ports
 - Detect OPEN/CLOSED ports
 - Accept hostname or IP address
 - Simple command-line interface

## Technologies
 - Python3
 - Linux
 - Socket Programming
 - Git
 - Github

## How to Run
python3 scanner.py
